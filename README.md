# training-web

## 介绍

培训系统前端项目，包含所有web端页面，页面跳转控制。通过ajax请求后端服务获取数据。

## 包含技术

- js框架：vuejs
- vue组件通信框架：vuex
- 前端路由框架：vue-router
- ajax框架：axios
- ui框架：element-ui
- vue脚手架：vue-cli
- es6转换框架：babel
- 打包框架：webpack
- js语法检查框架：eslint

## 安装步骤

1. 安装node.js
2. 安装cnpm
3. 下载本项目
4. 在项目路径下执行 cnpm install（解释：自动下载项目依赖）
5. 在项目路径下执行 cnpm run dev（解释：启动前端项目）
6. 在浏览器中输入 http://localhost:8081
